spark-bigquery
===============

This Spark module allows saving DataFrame as BigQuery table.

## Usage

```scala
import com.appsflyer.spark.bigquery.BigQuerySpark._

df.saveAsBigQueryTable("project-id:dataset-id.table-name")
```
